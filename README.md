## Getting Started

- ## Server

     * ####  Run
     	```bash
      	   # install
      	   $ bundle install

      	   # development
      	   $ rails server



     * #### Database Seeds
		```bash
		   $ rails db:migrate
       # to run the seed file (Learning-Managment-System-Backend/blob/main/db/seeds.rb)
		   $ rails db:seed 
		```
